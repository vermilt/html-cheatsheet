HTML Cheatsheet

Useful HTML tags

Tag Name Description 
<h1>
 Heading HTML Headings are titles / subtitles for a webpage 
<p>
 Paragraph Paragraphs are for the main content of a webpage 
<a>
 Link Links allow users to click their way from page to page 
<br>
 Break Creates a line break within text 
<strong>
 Strong Strong allows users to have bold text 
<em>
 Emphasis Emphasis allows users to have text in italics

Media tags

Tag Name Description 
<img>
 Image Images can improve the design and the appearance of a web page 
<audio>
 Audio The Audio element is used to play an audio file on a web page 
<video>
 Video The Video element is used to show video on a web page
